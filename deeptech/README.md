# DeepTech

## Installation (Linux/WSL)

Install my-project with npm

```bash
  pip install virtualenv
  git clone https://github.com/geoffreynyaga/deeptech.git .
  virtualenv venv
  source venv/bin/activate
  pip install -r requirements.txt
```
